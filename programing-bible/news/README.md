# 初始化


# 安裝 Django 模組
```shell
    pip install django==3.1.7
```

# 建立 Django 專案語法
```shell
    django-admin startproject news
```

# 執行伺服器
```shell
    cd news
    python manage.py runserver # windows
    python3 manage.py runserver # mac
```

# 建立 Application 應用程式
```shell
    python manage.py startapp newsapp # windows
    python3 manage.py startapp newsapp # mac
```
